# Ebury Fire Systems — Pre-launch website

This repository hosts the pre-launch holding site at https://eburyfire.co.uk

## Files

- `index.html` — the homepage (single-file site, all content)
- `CNAME` — tells GitHub Pages which custom domain serves this repo. Do not delete.
- `robots.txt` — search engine crawl instructions
- `sitemap.xml` — page index for search engines

## To update the site

1. Open `index.html` here on github.com
2. Click the pencil icon to edit
3. Make changes, scroll down, click Commit changes
4. Site updates at eburyfire.co.uk within 60 seconds

## To replace the placeholders in the footer

When Companies House registration completes, replace `[pending]` in the footer with the real company number, registered office and (later) VAT number.
